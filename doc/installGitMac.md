# Gitのインストール(Mac)

## 前提条件
* [Mavenのインストール](installMavenMac.md)が終了していること。

## インストール
[Terminalを起動](tipsForMac.md#terminalの起動方法)して

```sh
brew install git
```
してください。処理が終了した時に :beer: のアイコンが出ていたらOKです。

## インストールできたら
[Terminalを起動](tipsForMac.md#terminalの起動方法)して

```sh
git --version
git version 2.9.0
```
というように `git` コマンドが動くことが確認できればOKです。
